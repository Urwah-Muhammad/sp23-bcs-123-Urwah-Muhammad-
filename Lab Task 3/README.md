# Lab Task 3 - Express.js with EJS

## Installation

1. Install dependencies:
```
npm install
```

2. Start the server:
```
npm start
```

3. Open your browser and go to:
```
http://localhost:3000
```

## Project Structure

- `server.js` - Main Express server file
- `routes/` - URL routes
- `views/` - EJS templates
  - `partials/` - Reusable components (header, footer)
- `public/` - Static files
  - `css/` - Stylesheets
  - `images/` - Images
  - `js/` - JavaScript files
